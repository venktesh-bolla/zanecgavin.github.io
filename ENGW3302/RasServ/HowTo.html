Materials: What you'll need

Rasberry Pi

The obvious first thing you’ll need is a raspberry pi, the tiny computer on which your server will run. This will cost around $38 (as of Nov. 2015) on Amazon.

Micro SD Card

The operating system that we will be running the raspberry pi on will run natively from a micro sd card, like those in many android phones and small electronics. A 32 GB Micro SD only costs $10 on Amazon, or you can get a smaller one for even less. Note: It is recommended that the Micro SD be at least 8 GB.

USB Keyboard and Mouse

For the initial setup of the raspberry pi an usb keyboard and mouse will be plugged into two of the four usb ports on the pi. There are many other possible configurations – Bluetooth, SSH, VNC – that work equally well.

USB Keyboard - $12 Amazon

USB Mouse – $6 Amazon

USB WiFi Adapter

A wireless server is certainly not a requirement, just a matter of convenience. There are several WiFi adapters that work with Raspian out of the box, without any further configuration. The one I use is the Edimax Nano, which costs only $10 on Amazon.

USB Storage

One requirement for a file server is that there be some place to store the files, ideally you have a 1 TB USB HDD lying around, if not you can use any old flash drive. This one from Amazon is 128 GB and only costs $22. Note: Any USB Hard disk drives must be able to run on USB power supply.

Some other things to consider

There are some things that you probably already have, but you may need to purchase in order to complete this guide:

  • HDMI Monitor
  • HDMI Cord
  • Micro USB Cord (for power)

Step 1: Start with updating the distro

command sudo apt-get update

Step 2: Let's add users to the www-data group

command sudo usermod -a -G www-data www-data

Step 3: Next we'll install all the packages you will need, this will take some time

commandsudo apt-get install nginx openssl ssl-cert php5-cli php5-sqlite php5-gd php5-common php5-cgi sqlite3 php-pear php-apc curl libapr1 libtool curl libcurl4-openssl-dev php-xml-parser php5 php5-dev php5-gd php5-fpm memcached php5-memcache varnish

Step 4: This command will create and walk you through writing the basic SSL certificate

command sudo openssl req $@ -new -x509 -days 730 -nodes -out /etc/nginx/cert.pem -keyout /etc/nginx/cert.key

Step 5: Now change the privileges of the files you just referenced. This lets the file owner read and write to the file and is required for the next step.

command sudo chmod 600 /etc/nginx/cert.pem sudo chmod 600 /etc/nginx/cert.key

Step 6: Next, we’ll change the nginx default config file. You can copy and paste the text at this link (or copy the file).

command sudo nano /etc/nginx/sites-available/default

You will need to change one thing in this file. Replace the following with your raspberry pi’s ip address.

10.0.0.ip; #your raspberry pi's ip here

Note: If you’d like to access your server from somewhere outside your local network use your Global IP instead.

Step 7: Now, we’ll edit the php.ini files to allow files of 2GB

command sudo nano /etc/php5/fpm/php.ini

search for upload_max_filesize and set it to 2000M

search for post_max_size and set it to 2000M

Step 8: Next make the following two changes to the www.conf and dphys-swapfile file

command sudo nano /etc/php5/fpm/pool.d/www.conf

Change listen = /var/run/php5-fpm.sock to listen = 127.0.0.1:9000

command sudo nano /etc/dphys-swapfile

Change CONF_SWAPSIZE=100 to CONF_SWAPSIZE=512

Step 9: Reboot the pi

command sudo reboot

Step 10: Download and install owncloud, the server software, then clean up

command sudo mkdir -p /var/www/owncloud

command sudo wget https://download.owncloud.org/community/owncloud-8.0.2.tar.bz2

command sudo tar xvf owncloud-8.0.2.tar.bz2

command sudo mv owncloud/ /var/www/

command sudo chown -R www-data:www-data /var/www

command rm -rf owncloud owncloud-8.0.2.tar.bz2

Step 11: Now plug in your USB storage or drive.

Some of the next steps will vary depending on the USB file system of your device. This tutorial will include the steps for using FAT32 type file systems. Note: You may need to download the drivers for other filesystems, such as exfat, ntfs, etc…

Step 12: Make a directory on your pi where the drive will be mounted, and mount the drive (if there is only one drive it will likely be located at /dev/sda1 as indicated below)

command sudo mkdir /media/owncloud

command sudo mount -t vfat -o uid=1000,gid=1000,umask=007 /dev/sda1 /media/owncloud

Step 13: Next you will be editing the fstab file. If this is modified wrong the pi will boot into emergency mode, so first we’ll back it up

command sudo cp /etc/fstab /etc/fstab.backup

command sudo nano /etc/fstab

Add the following after the last row /dev/sda1 /media/owncloud vfat uid=1000,gid=1000,umask=007 0 0

This will mount the drive automatically on startup. Note: The pi will boot into emergency mode if it cannot find the disk added to fstab, so use caution. You can always edit fstab, even in emergency mode.

Step 14: Now we’ll wrap up and edit two local file to allow 2GB file sizes. Make the changes as listed

command sudo nano .htaccess

Set Php_value upload_max_filesize to 2000M

Set Php_value post_max_size to 2000M

Set Php_value memory_limit to 2000M

command sudo nano .user.ini

Set upload_max_filesize to 2000M

Set post_max_size to 2000M

Set memory_limit to 2000M

Step 15: From a computer on the same network go to the pi’s ip address in the browser and make an admin ownCloud account.

Your Network Attached Storage device is now working!

Tips: How to finish fast

There are some commands that will be useful to know if you are not well versed in terminal-speak

Find what disks are attached

command sudo fdisk -l

This will list your storage devices, the information most useful here will be at the very bottom of the information presented in the terminal.

For me

Device : /dev/sda1

Type: FAT32

Finding your ip address

Another thing you will want to know throughout this set-up is the wlan0 or eth0 ip address of you pi.

command ifconfig

You will use inet addr: from the terminal output, from wlan0 if your server will be wireless.
